# Plugins

## Other
_Plugins that don't fit in with any existing category._

- [WebServer Plugin](https://github.com/nitrado/hytale-plugin-webserver) - This plugin allows other Hytale plugins to serve web-based content.
- [Plugin Query](https://github.com/nitrado/hytale-plugin-query) - This plugin exposes server information via an HTTP endpoint, allowing services to access information about a server such as player list, etc.

## Performance
_Plugins for user permission management._

- [Plugin Performance Saver](https://github.com/nitrado/hytale-plugin-performance-saver) - This plugin ensures stability of a Hytale server by lowering the resource consumption of the server when it is under resource pressure.

## Region and World Management
_Plugins for managing world regions._

- [SimpleClaims](https://github.com/Buuz135/SimpleClaims) - Adds both parties and claims, allows players to claim regions that only their party can interact with.
